## 🎪 Problem Discovery Process

```bash
# 我的工具发现流程

1. 观察日常工作中的摩擦点
   $ grep "这应该有工具" ~/daily_notes.md

2. 验证痛点是否普遍
   $ search reddit --query "有没有工具能..."
   $ search v2ex --query "求推荐..."

3. 构建最小可行方案
   $ build mvp --minimal --focused

4. 快速验证价值
   $ deploy && measure adoption && iterate
```

**痛点来源**：
- 💼 工作中的重复劳动
- 🤔 "为什么没有..."的吐槽
- 📊 数据分析发现的高频问题
- 🗣️ 社区里的"求推荐"帖子
